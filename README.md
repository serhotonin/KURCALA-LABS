## 🌟 Proje Hakkında
**KURCALA (A.E.T.H.E.R. LABS)**, ortaokul düzeyindeki (5-8. sınıf) öğrenciler için fen bilimleri konularını soyut olmaktan çıkarıp interaktif birer deneyime dönüştüren dijital bir laboratuvar platformudur. Geleneksel öğrenme yöntemlerini modern pedagojik yaklaşımlar ve oyunlaştırma elementleri ile harmanlayarak öğrencilerin fen bilimlerine olan merakını ve başarısını artırmayı hedefler.

## 🚀 Öne Çıkan Özellikler

### 🧬 Pedagojik Araçlar
*   **Dijital Laboratuvar Defteri:** Öğrencilerin deney öncesi hipotez kurmalarını, deney sırasında ise bilimsel gözlem notları almalarını sağlayan interaktif bir araçtır.
*   **Senaryo Tabanlı Öğrenme:** Fen konuları "Görev Brifingleri" şeklinde sunulur. Öğrenci, sadece bir simülasyonu izlemek yerine "Buzul Gezegeni Görevi" gibi senaryolarda problemleri çözen bir bilim insanı rolünü üstlenir.
*   **Sandbox (Serbest Mod):** Müfredat kazanımları tamamlandıktan sonra açılan bu modda, öğrenciler fiziksel kuralların sınırlarını zorlayarak "Ya şöyle olsaydı?" sorusuna yanıt ararlar.

### 🎮 Oyunlaştırma ve Motivasyon
*   **Başarım ve Rozet Sistemi:** Hız, titizlik ve merak gibi farklı kategorilerde (Hız Tutkunu, Kaos Teorisyeni, Titiz Araştırmacı) dijital rozetler kazanılarak öğrenme süreci ödüllendirilir.
*   **İnteraktif Dashboard:** Öğrencinin çalışma süresini, tamamlanan modülleri ve başarı puanlarını anlık olarak takip edebileceği merkezi bir yönetim paneli.

### 🧪 Mevcut Simülasyonlar
*   **5-6. Sınıf:** Devre Simülasyonu, Karanlık Mahalle (Işık), Güneş Sistemi Keşfi.
*   **7-8. Sınıf:** Asit-Baz Dengesi, Simya Karışımları, Basınç Deneyleri, Hava Durumu Modelleme.

## 🛠 Teknik Altyapı
Modern web teknolojileri ile geliştirilen platform, yüksek performans ve kullanıcı dostu bir arayüz sunar:

*   **Frontend:** React 19, TypeScript, Vite
*   **Arayüz Tasarımı:** Material UI (MUI) - Modern, minimalist ve profesyonel tasarım
*   **Backend:** Node.js, Express API
*   **Veritabanı:** SQLite (better-sqlite3) ile kalıcı veri ve tercih yönetimi
*   **Sistem:** Karanlık/Aydınlık Tema desteği, Çoklu Dil desteği

## 📦 Kurulum ve Çalıştırma

### Gereksinimler
- Node.js (v18+)
- npm veya yarn

### Adımlar
1.  **Projeyi Klonlayın:**
    ```bash
    git clone https://github.com/serhotonin/KURCALA-LABS
    cd KURCALA
    ```
2.  **Bağımlılıkları Yükleyin:**
    ```bash
    npm install
    cd server && npm install
    cd ..
    ```
3.  **Uygulamayı Başlatın:**
    *   **Frontend:** `npm run dev`
    *   **Backend:** `node server/index.js`
